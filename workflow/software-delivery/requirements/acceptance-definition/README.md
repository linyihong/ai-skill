# Acceptance Definition

## 目的

將 behavior discovery 轉成 acceptance criteria 與 validation target。

## 步驟

1. 定義 happy path、negative path、edge case。
2. 選擇 validation target：unit、contract、BDD、fixture、manual evidence 或 release checklist。
3. 記錄 regression scope。
4. 確認 acceptance criteria 不包含未確認 feature。

## 完成條件

沒有 validation target 的 acceptance criteria 不能作為完成宣告基線。

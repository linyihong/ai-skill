# Platforms

Use this directory for app-type or platform-specific implementation guidance.

Prefer `controls/` for cross-platform principles, then link here for implementation differences.

| Directory | Scope |
| --- | --- |
| `mobile/` | Android, iOS, Flutter, React Native, and mobile release concerns. |
| `web/` | Browser/frontend app hardening. |
| `backend/` | API/server controls that mobile apps depend on. |

# Mobile Platforms

Use this directory for mobile-specific guidance.

| File | Scope |
| --- | --- |
| `android.md` | Native Android, Kotlin/Java, manifest, network security config, release build concerns. |
| `ios.md` | iOS, Keychain, ATS, jailbreak risk signals, release configuration. |
| `flutter.md` | Flutter/Dart apps, platform channels, Dart AOT, plugin boundaries, release checks. |
| `react-native.md` | React Native bridge, JS bundle exposure, native modules, release checks. |

Cross-platform controls still live in `../../controls/`.

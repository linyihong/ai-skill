# dynamic_resize

Use when the usable viewport can change after initial render.

## Governs

- Browser chrome expansion or collapse.
- Orientation changes.
- Split-screen or resizable windows.
- Developer tools device emulation changes.
- Reload after viewport changes.
- Keyboard or system overlay effects when relevant.

## Fixture Guidance

Validate both fresh load and resize-then-reload when the defect report involves viewport switching or stale layout measurements.

# large_mobile

Use when wider phone-like screens or compact phablet layouts may change density and spacing assumptions.

## Governs

- Card and grid density.
- Max-width caps.
- Centered shell behavior.
- Fixed navigation width and alignment.
- Content hierarchy between mobile and tablet layouts.

## Fixture Guidance

Use this context when a UI has phone-specific navigation but enough width to tempt multi-column or denser layout decisions.

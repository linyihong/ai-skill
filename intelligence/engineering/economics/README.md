# Engineering Economics

Engineering economics captures cost models, timing effects, and investment trade-offs. These are not heuristics by themselves; they explain why a decision becomes cheaper or more expensive.

## Models

| Model | Use When |
| --- | --- |
| [`mobile-first-economics.md`](mobile-first-economics.md) | A UI surface has mobile or constrained render-context obligations. |
| [`responsive-cost-curve.md`](responsive-cost-curve.md) | Deciding whether a responsive issue should be handled early, deferred, or escalated. |

## Boundary

- Economics explains cost pressure.
- Heuristics provide decision shortcuts.
- Workflow defines execution order.
- Governance defines authority, severity, and evidence expectations.
